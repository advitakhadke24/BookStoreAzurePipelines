# BookStore
This project was used as an example of how to deploy a .NET 6 Web API with a SQL database by using Azure Pipelines

## Technologies
- .NET 6
- Entity Framework 6
- Fluent API
- AutoMapper
- Swagger
- SQL Server

### Unit tests
- xUnit
- Moq
- SQLite In-Memory database

## Architecture
- 3 Layers:
  - Application layer (API)
    - Controllers
    - Dtos
  - Domain layer
    - Models
    - Interfaces
    - Services
  - Infrastructure layer
    - Repository Pattern
